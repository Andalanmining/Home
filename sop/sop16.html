Prima Nirbhaya Management System Element 2.06

STANDARD OPERATING PROCEDURE (SOP 16)
PRE START CHECK

1.0 Tujuan

Prosedur ini dibuat dengan tujuan untuk memberikan petunjuk tentang langkah-langkah yang harus dilakukan pada saat melakukan pemeriksan awal (pre start check) pada kendaraan / alat sebelum digunakan, untuk memastikan kelayakan pakai.

2.0 Lingkup

Berlaku untuk pemeriksaan awal pada kendaraan / alat sebelum dioperasikan.

3.0 Tanggung Jawab

  • Driver / Operator
  • Supervisor

Pra syarat: Memiliki kimper valid, memahami SOP.
PPE: Sepatu safety, helm, sarung tangan.

4.0 Prosedur

  1. Persiapan – Parkir di tempat aman, pasang ganjal roda & cone.
  2. Pemeriksaan Sekeliling – Cek kondisi ban, track, chasis, lampu, tie rod, tekanan ban, bucket (excavator), blade (dozer).
  3. Pemeriksaan di Atas Kendaraan / Alat – Cek battery, oli, minyak rem, radiator, V-belt.
  4. Pemeriksaan di Kabin – Cek sabuk pengaman, lampu indikator, steering, rem, PTO, handle, perlengkapan tambahan.

Gunakan checklist dan catat semua temuan. Jika ditemukan kerusakan pada kemudi, rem, atau sabuk pengaman, kendaraan/al alat tidak boleh dioperasikan.

5.0 Bagan Alir

Tidak ada.

6.0 Referensi

FPE 2.19 Kondisi Kendaraan dan Alat Berat Bergerak
